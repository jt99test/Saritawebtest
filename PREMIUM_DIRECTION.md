# Premium Direction

## Direction
Sarita should feel like a dark editorial astrology observatory: intimate, cinematic, precise, and calm.

The premium feel should come from restraint:
- fewer framed image blocks
- full-bleed or unframed hero moments
- tighter section rhythm
- soft local texture instead of loud space wallpaper
- ivory type, antique gold accents, and quiet violet depth
- precise chart geometry as the most technical visual object

## Avoid
- cute mystical clip-art
- random decorative sparkles
- giant empty poster gaps
- framed hero images with CTA underneath
- saturated purple glow everywhere
- every section using the same centered layout

## Image Priorities
Best assets to send or replace next:

1. Home hero reference
   A wide cinematic image that feels like an expensive astrology editorial cover. It should support text over it and not contain embedded words.

2. Moon assets
   New moon and full moon should feel like one matched pair: same lighting, same crop logic, same texture level.

3. Element assets
   Fire, earth, air, and water should be a matched set, ideally abstract but tactile. They should not look like separate AI generations from different worlds.

4. Portrait / brand mood references
   If Sarita has a human/personality direction, send 2-4 references for the feeling: elegant, warm, mystical, clinical, sensual, etc.

## Layout Rule
Each screen should have one dominant visual moment, then the rest should read like a refined tool or editorial report. Do not make every section compete for cinematic drama.
